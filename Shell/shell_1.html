Introduction

The shell is a command-line interface that run programs and shows the programs output; it is commonly used because of its speed and flexability. The shell is similar to programming languages in the same sense that they use special command words to function. Programs that run in the shell are known as terminals. The shell is the outermost layer of an operating system and allows you to interact with the tools of the operating system. There are various types of shells; one of the more commonly used ones is BASH. (Unix shell)

There are two types of shells: graphical shells and text shells. Graphical shells are ones where you use boxes, windows, and buttons to interact with the operating system. This type of interface is also known as a Graphical User Interface or GUI Text shell, by contract, use typed commands. Text shells are also known as Command-line interface or CLI

To run commands in BASH(and several other shell langauges) simply type the command you want and hit ENTER At the botton of this page is a list of shell commands. It is worth noting that there are some special characters(like !) that have a special meaning. To avoid uses these symbols unintentionally, you can use ''.

Navigating Directories

Directories are like the folders you see when you use your computer(like the download folder.) The shell has a way of listing the content of the directory via the ls command. ls means list. Normally when you open the shell, the shell will start off in your home directory. To list off the name of a different directory use ls followed by the directories name. To change the directory that the shell is looking at you can use the command cd which stands for change directory. Cd can also be used to go to another directory that is not directly inside the current directory, use the command followed by all the steps you need to get to the directory you want(going down); this is known as a relative path. If you want to access any file located anywhere, you can type out the full path to get to that file starting from the home directory; this is known as an absolute path.
If you want to go back a directory(aka parent directory) use the command ... In case you get lost, you can use the pwd (print working directory) command and it will print the location direction. Working directory are the current directory. The unix system uses forward slashes not backslashes to seperating names in directory. The result of the pwd that is seperated by slashes is called a path. The home directory is the top-level directory(has all the user directories inside it)
