Tags

inline element

    1. em = emthesis (italics)
    2. strong = bold
    3. mark = highlight
    4. sub = move text down vertically slightly (subscript) like chemical elements
    5. sup = move text up vertically slightly (superscript) like power sign
    6. a = anchor(link to another page). The a must be followed with attribute href = "url" and content(which will be the way the link appears.

block elements

    1. p = paragraphs.
    2. h1 - h6 = headline or heading in document. Creates margin around itself
    3. ul = unorganized list. List content with bullet point. Uses element li to create list.
    4. ol = organized list. List content with numbers by default but can also use letters. Uses element li to create list.
    5. div = divide. Used to divide and organize content on a page. Heavily used to customize the style with CSS

void element or empty elements

inline element

    1. br = inserts line break
    2. img = image. Refers to an image and displays that image. Has two attributes, src and alt.
