Tags

inline element

    1. em = emthesis (italics)
    2. strong = bold
    3. mark = highlight
    4. sub = move text down vertically slightly (subscript) like chemical elements
    5. sup = move text up vertically slightly (superscript) like power sign

block elements

    1. p = paragraphs.
    2. h1 - h6 = headline or heading in document. Creates margin around itself
    3. ul = unorganized list. Uses element li to create list.
    4. div = divide. USed to divide and organize content on a page.
