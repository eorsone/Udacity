A program is a set of instructions, written on lines of code, for a computer to follow. Each line is an instruction for the computer to follow.

Variable

When running a program, the computer will find some free space on the computer and put the data of the program into a space. This prepares the program to run. When we give the program, stored in memory, a named by adding a (name) = code it is known as a variable. A variable is similar to a label/nametag. When connecting a name to data in memory it is known as assignment.

Strings

String are anything that is written in quotes. This how programmers and python identify what is a string. for example amy.color("white"), white is a string. Without the quotes, the program will not run.

Lists

List are anything in python that is surrounded by [].

Comments

Any line that starts with a # symbol is a comment. Comments have no direct effect on the code; rather, they are left behind for humans to read. Using comments can allow the program to be more easily understood, as in defining what a certain piece of code does or why it is there. Commenting can also be used to comment-out code to try and fix problem and test a certain part of a code, but they are normally removed before the final draft. On mac and window there is a shortcut command for most code editors that automatically toggle comments on and off.

  • Mac - command + /
  • Window - ctrl + /

Looping

To reduce repeating, it is best to use something known as a loop. One type of loop is known as for loop. For loops look like this for ____ in [________](list): followed by a list of indented commmands. This commands runs the same commands as many times as there are objects in the list. It is worth mentioning that the list can be used as an input for a command. This is known as looping over the list.
