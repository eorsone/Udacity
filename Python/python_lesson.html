A program is a set of instructions, written on lines of code, for a computer to follow. Each line is an instruction for the computer to follow.

Variable

When running a program, the computer will find some free space on the computer and put the data of the program into a space. This prepares the program to run. When we give the program, stored in memory, a named by adding a (name) = code it is known as a variable. A variable is similar to a label/nametag. When connecting a name to data in memory it is known as assignment.

Strings

String are anything that is written in quotes. This how programmers and python identify what is a string. for example amy.color("white"), white is a string. Without the quotes, the program will not run.

Lists

List are anything in python that is surrounded by [].

Errors

Errors are bound to happen in programming. Luckily, most programming languages have a way of identifying error. In python, error messages will appear giving a brief discription of why there is an error, and where it is located(in numbers). Sometimes the line number the error message gives you will not be 100% accurate, but it should bring you relatively close to the location.

  • Name error = code is refering to a name that has not been define yet.
  • Syntax errors = when code doesn't follow the grammar of the programming language
  • Usage error = when your commands don't make sense with what you want to do. example - alison.forward("orange"
  • Logic error = error created by the user and not the compueter; an undesired outcome.

Modules

Modules are define as A file that has a collection of useful code that can be used in other python programs. To use a modules, you must include an import____ command at the top of the code. In the code below, turtle.Turtle(), the first turtle before the . is calling upon the module that we imported. The . tells the program to look for the code named Turtle inside the turtle module. Comments

Any line that starts with a # symbol is a comment. Comments have no direct effect on the code; rather, they are left behind for humans to read. Using comments can allow the program to be more easily understood, as in defining what a certain piece of code does or why it is there. Commenting can also be used to comment-out code to try and fix problem and test a certain part of a code, but they are normally removed before the final draft. On mac and window there is a shortcut command for most code editors that automatically toggle comments on and off.

  • Mac - command + /
  • Window - ctrl + /

Looping

To reduce repeating, it is best to use something known as a loop. One type of loop is known as for loop. For loops look like this for ____ in [________](list): followed by a list of indented commmands. This commands runs the same commands as many times as there are objects in the list. It is worth mentioning that the list can be used as an input for a command. This is known as looping over the list. Loop can also be nested within loops(nested loop) by just indenting.

Statements

Statements are lines of text that give the computer some instruction to carry out. Statements can vary such as assignment statement, import statements, and call statements(these are all simple statements.) Simple statements are used to control whether, when, and how many times a code runs from top to bottom. If the statement has other statements inside of it, it is known as a compound statement.(for loop) Compound statements are less restricted in the order in which the code is run; for example, in a for loop, the loop is repeated several times before continuing down the line of code. The order in which the statements are executed is known as the control flow Since compound statements are made up of simple statements, they can also control whether, when, and how many times a code runs.

Note

The range function is a easy way to get a certain number of items on the list starting from 0. For example if we typed sides = range(4), this would be like typing sides = [0,1,2,3]

Crunching numbers

Computers are able to do math just like a calculator; in coding these are known as arithmetic expressions

  • addition - uses the + sign
  • subtraction - uses the - sigh
  • multiplication - uses the * sign
  • division - uses the / sign
Expression is a piece of code that results in some value. It is also known as evaluation. In computer science,operators are the ones that tell us what operation we are preforming( + - * /), while operand are the values being operated on(anywhere where you can use a number in code, you can replace it with a arithmetic expression) You can use an arithmetic expression and store it as a variable.

Function

A call statement(call) is a statement that, when it runs, executes a block of code. It can also take in the input that the block of code needs to run, this is known as passing(giving input to a function.) An argument is the information being passed to a function. The function can then return an output of some kind. For example:
for side in range(100) - This line is calling the range function and passing it the argument100.
Note - A callable code is a code that will only run when a call statement is used.
Note - A function is a block of code that can be called and it does some task.

Method Call vs Function Call

A method is a piece of code that is associated with a particular object(like a variable). A function is nonspecific. A method is a subset of function. All methods are functions but not all functions are methods.

Variable scope

A scope is a part of the code for which a variable is defined. If a variable is define inside of a function, in can only be used inside the function(these are known as local scopes.) A global scope is one that is defined outside of the function and can be used anywhere else in the file.
