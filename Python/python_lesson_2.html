Defining Functions

One good reason to define your own function is to prevent repetition of a useful piece of code that you may want to use several times in your program. To define a function:

  1. Write the words def(stands for define) followed by the name of the function
  2. Indent the next line so everything written will be inside the function
  3. Put your code inside the function
Keep in mind that the function will not work unless you call it. TO call the function, simply write the functions name followed by ().

Parameters and Arguments

To allow the function to accept input, we must do it on the same line we define the function; more specifically, we place a variable between the ().(this is similar to an assignment statement and is known as parameters.) If you want more than one parameter, simply add a comma(,) after the first parameter(you would also use this same method when you call upon to arguments) The assignment goes in the order in which the parameters were written; for example
def something(x,y)
something(20,10)
In this example x gets assigned 20 and y gets assigned 10

If Statements

If statments are compound statements that run only if a certain condition is met. If statements look like this: if number == 4:(followed by the command you want to run. Note that there is a ==(equality operator) instead of =; if there was just a =, it would be an assignment statement. You can also add an else statement which allows the run a seperate command for when the if statement isn't true.

Modulo

The Modulo is related to , in the sense that if you divide, you sometimes have a remainder, that remainder is the modulo. Modulo are good for functions involvoing cyles or a repeated pattern of commands. If you set the module to x % y, the result will always be less than y before and will go up by 1 each time until the result equals y-1

Returns

Returns can be placed at the end of the function for it to return a value. In the example below, the function times_ten takes in the number 5 and return the number 50 which is then assigned to variable distance.
Return can move a value from a function back to the one that calls it. There can be more than one return statement; however, the function can only return one return statement(and will always return the first return statement it finds.)

Random Module

A deterministic program: a program that always produes the same output for a given input
There are module that allows you to randomize a selection in python they are

  • random.randint - takes the lowest and highest integer and randomly select the numbers between that number
  • random.choice - take a list and returns a randomly chosen items from the list.
Note: you must import random into your document before you can use the random module.
Note: Don't forget the module name.

Comparison Operators

You may want to use operators to compare numbers or set certain conditions, here are a few operation you can use in python

  • a == b - is a equal to b?
  • a < b - is a less than b?
  • a > b - is a greater than b?
  • a <= b - is a less than or equal to b?
  • a >= b - is a greater than or equal to b?
  • a != b - is a not equal to b?
All of these can be used in if statements

True or False

If statements always contain a condition such as 1 < 2. For the statement, python must decide wheither the statement is true or false. Python uses the values "True" and "False" if the statement is true, python evaluate to the "True" value, likewise if the statement is false, python will evauluate to the "False" value. You can use these values in an if statement and also assign these values to a variable.
